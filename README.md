Developed a full business website for a solar energy company using HTML5, CSS3, and JavaScript. 
The project focuses on responsive design, ensuring a smooth experience across both desktop and mobile devices. 

Key features include dynamic content loading, interactive product modals with detailed specifications, form validation with email integration, and an image gallery with lightbox support.

The site also includes performance optimizations and security measures such as input sanitization, CSRF protection, and rate limiting. Additionally, an SEO-friendly structure was implemented with robots.txt and sitemap.xml to improve search engine visibility.
